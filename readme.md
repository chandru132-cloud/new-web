## 📖 Project Description

**My Mobile Store** is a simple HTML website that displays information about popular smartphone brands such as **iPhone, Samsung, and OnePlus**.
The website is designed as a beginner web development project using **HTML5**. It provides descriptions and images of different mobile phones and includes a basic navigation menu to explore different sections.

This project helps beginners understand how to structure a webpage, add images, create links, and organize content using HTML.

## ✨ Features

* Simple and clean mobile store webpage
* Navigation menu for different brands
* Sections for **iPhone, Samsung, and OnePlus**
* Mobile images displayed for each brand
* Contact information section
* Beginner-friendly HTML structure

## 🛠 Technologies Used

* **HTML5**
* Basic webpage elements (headings, paragraphs, images, links)
* Simple layout using HTML tags

## 🚀 How to Run the Project

1. Download or clone this repository.
2. Open the project folder.
3. Locate the **index.html** file.
4. Double-click the file or open it in a web browser.
5. The Mobile Store website will load.
