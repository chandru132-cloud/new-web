# 📱 My Mobile Store Website

## Project Description

My Mobile Store is a simple HTML website that displays information about popular smartphone brands such as iPhone, Samsung, and OnePlus. The website provides basic details about each brand along with images of the phones. It is designed as a beginner-level web development project using only HTML.

## Features

* Simple and clean webpage design
* Navigation links for different mobile brands
* Information about iPhone, Samsung, and OnePlus
* Mobile images displayed on the page
* Contact information section
* Beginner-friendly HTML structure

## Technologies Used

* HTML5
* Basic webpage structure (head, body, headings, paragraphs)
* Images and hyperlinks

## Project Structure

* index.html – Main webpage
* iphone.html – iPhone page
* samsung.html – Samsung page
* oneplus.html – OnePlus page
* images – Folder containing mobile images

## How to Run the Project

1. Download or clone the project.
2. Open the project folder.
3. Double-click the **index.html** file.
4. The website will open in your web browser.
